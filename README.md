# Proyecto 1 de Backend 

Sistema de biblioteca virtual en donde los usuarios pueden hacer diferentes funciones.

## Características

- **Registro y Autenticación**
- **Gestión de Usuarios**
- **Gestión de Libros**
- **Historial de Reservas**
- **Filtros de Búsqueda**
- **Control de Permisos**

## Si se desea ejecutar el proyecto:

   ```bash
   git clone 
   ```
   ```bash
   npm install
   ```
   ```bash
   npm run dev
   ```
